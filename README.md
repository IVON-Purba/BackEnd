# Purba BackEnd
이 프로젝트는 Spring Boot를 기반으로 하며, RestFul API를 통해 데이터를 처리합니다. 데이터 저장소로는 MySQL과 Redis를 사용하며, AWS를 통한 클라우드 기반 인프라 설계로, 안정적인 서비스 운영을 목표로 합니다.



## 기술 스택

- **Spring Boot**
- **RestFul API**
- **MySQL**
- **JWT (JSON Web Token)**
- **Redis**: 키-값 저장소, 사용자의 Refresh Token 저장 용도
- **AWS**
